Webhook Payload (POST)
{
  "ngo_name": "Helping Hands",
  "people_count": 50,
  "location": "Hadapsar, Pune",
  "contact": "+91 98765 43210",
  "food_preference": "vegetarian"
}
Response Format
{
  "matched": true,
  "restaurant": "Spice Garden",
  "food": "Rice and Dal",
  "quantity_kg": 15,
  "location": "Koregaon Park",
  "contact": "+91 ...",
  "message": "Great news! ..."
}
Setup Steps in Botpress
  1. Create a new bot in Botpress Cloud and open the Flow builder
  2. Add an Execute Code node. Collect NGO name, people count, location, contact via bot messages
  3. In the Execute Code node, make a POST request to your backend URL + /botpress-webhook with the collected data
  4. Store response in a flow variable: workflow.matchResult
  5. Add a Text node that says: {{workflow.matchResult.message}}
  6. If matched = false, route to a "We'll notify you" message node
  7. Deploy your FastAPI backend publicly (use ngrok for testing or deploy to Railway/Render)
Botpress Execute Code Sample
const response = await axios.post(
  'https://your-api.com/botpress-webhook',
  {
    ngo_name: workflow.ngoName,
    people_count: workflow.peopleCount,
    location: workflow.location,
    contact: workflow.contact,
    food_preference: workflow.preference
  }
);
workflow.matchResult = response.data;
